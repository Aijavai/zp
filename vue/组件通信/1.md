# 组件通信

## props/$emit

父-》子  props 传变量
子-》父  props 传函数

自定义事件，父中定义，子中触发。

kebab-case 写法

## mitt

npm i mitt  // 安装，mitt 是一个工具

import mitt from 'mitt'

const emitter = mitt()
export default emitter
// emitter 可以绑定事件、触发事件

接收数据必定 绑定事件（发布）
发送数据必定 触发事件（订阅）

onUnmounted() => {}  在组件卸载时解绑事件 emitter.off()
