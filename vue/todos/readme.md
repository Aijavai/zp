# Todos 任务清单 VUE开发思路

- 传统做法
  先找到目标元素，然后对其做修改(DOM) 
  先找到DOM 元素 命令式，机械的，性能差的
    js(v8 快) -> html(渲染引擎 慢) , vue 底层也做, 代替我们做
    vue 让新手写的代码质量更好，容易入门 v-for {{}} ref
    js 操作dom 不仅仅是引擎完成 还包括渲染
- VUE 做法
  不在需要思考页面的元素怎么操作，而是要思考数据是怎么变化的。
- v-for 循环输出
  v- 指令 
- computed 计算属性
  - 性能更好，依赖的数据变化时才会重新计算
